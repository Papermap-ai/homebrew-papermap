# homebrew-papermap

Official Homebrew tap for Papermap.

## Install

Direct install (one command):

```bash
brew install papermap-ai/papermap/papermap
# alias
brew install papermap-ai/papermap/tui
```

Or tap once, then install by short name:

```bash
brew tap papermap-ai/papermap
brew install papermap
```

## Upgrade

```bash
brew update
brew upgrade papermap
```

## Uninstall

```bash
brew uninstall papermap
brew untap papermap-ai/papermap
```

## Maintenance

`Formula/papermap.rb` is published automatically by GoReleaser from
`Papermap-ai/papermap-tui` releases.
